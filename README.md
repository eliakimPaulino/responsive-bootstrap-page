# Layout Responsivo com Bootstrap 5

Projeto desenvolvido para praticar conceitos fundamentais de responsividade utilizando Bootstrap 5.

## Objetivo

Criar uma interface responsiva aplicando:

- Grid System
- Breakpoints
- Responsividade
- Utilitários do Bootstrap
- Organização de layout moderno

---

## Tecnologias Utilizadas

- HTML5
- Bootstrap 5

---

## Estrutura do Projeto

O projeto contém:

- Header responsivo
- Main com sistema de grid
- 3 colunas adaptáveis
- Footer centralizado
- Utilização de:
  - `container`
  - `row`
  - `col`
  - `shadow`
  - `rounded`
  - `text-center`
  - `bg-*`
  - `p-*`

---

## Responsividade

O layout foi construído utilizando os seguintes breakpoints:

```html
col-12 col-md-6 col-lg-4
```

### Comportamento

| Dispositivo | Layout |
|---|---|
| Mobile | 1 coluna |
| Tablet | 2 colunas |
| Desktop | 3 colunas |

---

## Conceitos Praticados

- Mobile First
- Layout responsivo
- Grid System do Bootstrap
- Experiência do usuário em diferentes dispositivos
- Organização visual com utilitários CSS
